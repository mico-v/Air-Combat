## 💡 使用建议

### 对于完全零基础的学习者

1. **第一阶段：环境准备**
   - 仔细阅读 `SETUP.md`
   - 按步骤安装 Visual Studio
   - 确保能编译运行测试项目

2. **第二阶段：理解框架**
   - 阅读 `README.md` 了解项目全貌
   - 阅读 `docs/02-框架概览.md` 理解架构
   - 阅读 `src/util/config.h` 了解配置参数
   - 阅读 `src/util/type.h` 了解数据结构

3. **第三阶段：学习API**
   - 把 `docs/API文档.md` 作为参考手册
   - 不需要记住所有函数
   - 需要时查阅即可

4. **第四阶段：实践编码**
   - 跟随 `GuideBook.md` 一步步实现
   - 遇到问题查看 API 文档
   - 善用调试工具

### 对于有编程基础的学习者

1. 快速浏览 `README.md` 和 `SETUP.md`
2. 重点阅读 `docs/02-框架概览.md` 了解架构
3. 直接开始编码，需要时查看 API 文档
4. 可以尝试添加自己的功能
